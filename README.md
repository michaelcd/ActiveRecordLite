#ActiveRecord Lite

An ORM built to duplicate the methods of Ruby on Rails' ActiveRecord. Abstracts complex SQL queries to Ruby objects via modules and inheritance.


## Methods
* ::all
* ::find
* ::where
* ::belongs_to
* ::has_many
* ::has_many_through
* #save
* #update
* #delete
* #destroy
